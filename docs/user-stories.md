# User Stories — AbhiIterates.OS

Format: "As a [role], I want to [action] so that [benefit]."
Priority: P0 = MVP blocker | P1 = MVP target | P2 = Phase 2 | P3 = Phase 3

---

## Authentication (US-001 to US-010)

US-001 | P0
As a new student, I want to register with my email and password so that I can create a personal account on the platform.

US-002 | P0
As a registered student, I want to log in with my email and password so that I can access my private library and study data.

US-003 | P0
As a student, I want to sign in with my Google account so that I do not have to remember a separate password.

US-004 | P0
As a logged-in user, I want my session to stay active across browser tabs so that I do not have to log in every time I open a new tab.

US-005 | P0
As a user, I want my session to expire after 30 days of inactivity so that my account remains secure on shared devices.

US-006 | P0
As a logged-in user, I want to log out so that my account is protected when I am done using the platform.

US-007 | P1
As a user who forgot my password, I want to receive a password reset email so that I can regain access to my account.

US-008 | P1
As a security-conscious user, I want to see all active sessions and revoke them so that I can remove access from devices I no longer use.

US-009 | P2
As a user, I want to enable two-factor authentication so that my account has an additional layer of security.

US-010 | P0
As a creator who wants to sell notes, I want to upgrade my account to Creator status so that I can list resources on the marketplace.

## Dashboard (US-011 to US-020)

US-011 | P0
As a student, I want to see a personalized dashboard when I log in so that I can immediately resume my study session.

US-012 | P1
As a student, I want to see how many hours I have studied this week so that I can track my consistency.

US-013 | P1
As a student, I want to see the resources I recently opened so that I can continue reading without searching.

US-014 | P1
As a student, I want to see my most recent AI chat summary so that I can recall what I was working on.

US-015 | P1
As a student, I want to see my saved bookmarks on the dashboard so that I can jump to important pages quickly.

US-016 | P2
As a student, I want to see a study streak counter so that I am motivated to study every day.

US-017 | P2
As a student, I want to see a weekly study analytics chart so that I can identify my productivity patterns.

US-018 | P1
As a student, I want quick-action buttons on the dashboard so that I can upload a file or open the AI without navigating the sidebar.

US-019 | P1
As a student, I want a "Continue Reading" section on the dashboard so that I can resume a PDF from the exact page I left off.

US-020 | P2
As a student, I want to create and track study tasks on the dashboard so that I have a simple to-do system without switching apps.

## Library & Resource Management (US-021 to US-035)

US-021 | P0
As a student, I want to create semesters in my library so that I can organize resources by academic term.

US-022 | P0
As a student, I want to create subjects inside a semester so that my notes are grouped by course.

US-023 | P0
As a student, I want to upload a PDF to a subject so that I can store and access it on the platform.

US-024 | P0
As a student, I want to view all resources inside a subject in a grid or list view so that I can visually browse my material.

US-025 | P1
As a student, I want to search for a resource by name so that I can find it without browsing every folder.

US-026 | P1
As a student, I want to filter my resources by type (PDF, Notes, Video) so that I can narrow my search.

US-027 | P0
As a student, I want to delete a resource I no longer need so that my library stays clean.

US-028 | P1
As a student, I want to rename a resource so that I can correct naming mistakes after uploading.

US-029 | P1
As a student, I want to move a resource from one subject to another so that I can reorganize without re-uploading.

US-030 | P2
As a student, I want to tag resources with custom labels so that I can find related material across subjects.

US-031 | P2
As a student, I want to mark resources as favorites so that my most important files are always accessible.

US-032 | P2
As a student, I want to archive old semester resources so that they are hidden but not permanently deleted.

US-033 | P1
As a student, I want to see the last-opened date on every resource card so that I know how recently I reviewed it.

US-034 | P1
As a student, I want to sort resources by name, date added, or date modified so that I can find things in the order I prefer.

US-035 | P2
As a student, I want to drag and drop resources into different subjects so that reorganizing is fast and intuitive.

## PDF Workspace (US-036 to US-050)

US-036 | P0
As a student, I want to open a PDF in a full reading view so that I can read it without leaving the platform.

US-037 | P0
As a student, I want to navigate PDF pages with Previous / Next buttons so that I can move through the document.

US-038 | P0
As a student, I want to jump to any page by typing the page number so that I can navigate large documents quickly.

US-039 | P0
As a student, I want to highlight text in any color so that I can mark important content visually.

US-040 | P0
As a student, I want my highlights to be saved automatically so that they persist when I close and reopen the PDF.

US-041 | P0
As a student, I want to bookmark any page so that I can return to it later.

US-042 | P0
As a student, I want to see all my bookmarks in a sidebar so that I can navigate to them with one click.

US-043 | P1
As a student, I want to add a text note to any page so that I can write my own thoughts alongside the content.

US-044 | P1
As a student, I want to see all my highlights in a summary sidebar so that I can review them without scrolling the entire PDF.

US-045 | P1
As a student, I want to zoom in and out of the PDF so that I can read small text comfortably.

US-046 | P1
As a student, I want to use keyboard shortcuts (J/K for pages, H to highlight) so that I can read without using my mouse.

US-047 | P1
As a student, I want to search for a word inside the PDF so that I can find a specific topic without reading every page.

US-048 | P0
As a student, I want to open an AI chat panel alongside the PDF so that I can ask questions about what I am reading without leaving the view.

US-049 | P1
As a student, I want to select text and immediately send it to AI as a question so that I can quickly clarify any confusing content.

US-050 | P2
As a student, I want to see the PDF's table of contents in the sidebar so that I can navigate by chapter.

## AI Workspace (US-051 to US-062)

US-051 | P0
As a student, I want to ask a question about the PDF I am reading so that the AI can answer using the document's content.

US-052 | P0
As a student, I want the AI to cite the page number where it found the answer so that I can verify the information myself.

US-053 | P0
As a student, I want to generate a summary of the entire PDF with one click so that I can get an overview before reading in depth.

US-054 | P1
As a student, I want suggested prompts to appear when I open the AI panel so that I know what kinds of questions to ask.

US-055 | P1
As a student, I want AI responses to render in formatted text (headings, bullet points, bold) so that they are easier to read.

US-056 | P1
As a student, I want to see AI responses stream in real-time so that I do not wait looking at a blank screen.

US-057 | P1
As a student, I want to view my previous AI conversations for a document so that I can reference past sessions.

US-058 | P2
As a student, I want to export an AI conversation as a PDF or text file so that I can save the study notes it generated.

US-059 | P2
As a student, I want to generate MCQs from the PDF so that I can test my understanding.

US-060 | P2
As a student, I want to generate flashcards from my highlights so that I can review key concepts quickly.

US-061 | P0
As a student, I want the AI to tell me when it does not know the answer rather than guessing so that I can trust the responses.

US-062 | P1
As a student, I want a daily AI usage limit so that I understand how much I can use before needing to upgrade.

## Marketplace (US-063 to US-075)

US-063 | P0
As a student, I want to browse the marketplace by subject category so that I can find notes relevant to my course.

US-064 | P0
As a student, I want to see a resource preview before buying so that I can judge quality before paying.

US-065 | P0
As a student, I want to purchase a resource with a single payment so that I can unlock access immediately.

US-066 | P1
As a student, I want to filter marketplace resources by price, rating, and subject so that I can narrow my search efficiently.

US-067 | P1
As a student, I want to read reviews from other students before buying so that I can trust the quality.

US-068 | P1
As a student, I want to see my purchase history so that I can re-access resources I have already bought.

US-069 | P2
As a student, I want to rate and review a resource I have purchased so that I can help other buyers make better decisions.

US-070 | P2
As a student, I want to save a resource to a wishlist so that I can buy it later.

US-071 | P0
As a creator, I want to list a PDF for sale with a title, description, price, and subject so that students can find and buy it.

US-072 | P0
As a creator, I want to receive payment in my bank account for every sale so that I earn from my content.

US-073 | P1
As a creator, I want to see how many times each resource was viewed and purchased so that I know what content is popular.

US-074 | P1
As a creator, I want to see my total earnings by month so that I can track my income.

US-075 | P0
As an admin, I want to review and approve or reject creator resource listings so that low-quality content does not reach students.

## Collaboration (US-076 to US-082)

US-076 | P2
As a student, I want to share a resource with a specific friend by email so that we can study the same material together.

US-077 | P2
As a student, I want to see my friend's highlights in real-time on a shared PDF so that we can annotate collaboratively.

US-078 | P2
As a student, I want to add a comment on a PDF page that my collaborators can see so that we can discuss content together.

US-079 | P2
As a student, I want to set permissions on shared resources (view-only or full-edit) so that I control what collaborators can do.

US-080 | P2
As a student, I want to see an activity timeline of changes made to a shared resource so that I know who did what.

US-081 | P3
As a student, I want to join a live voice session with collaborators while reading a shared PDF so that we can discuss in real-time.

US-082 | P2
As a student, I want to leave a shared resource without losing my own highlights so that my personal annotations are preserved.

US-083 | P1

US-084 | P1
As a creator, I want to receive a notification when someone purchases my resource so that I know when I earn money.

US-085 | P2
